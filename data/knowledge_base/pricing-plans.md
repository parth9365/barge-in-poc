# NovaBoard - Pricing Plans

## Overview

NovaBoard offers three pricing tiers designed to scale with your team. All plans include a 14-day free trial with full access to all features. No credit card required to start.

## Free Plan

**Price**: $0 / forever

Best for individuals and small teams getting started with project management.

### Included Features
- Up to 5 team members
- 3 active projects
- Kanban and List views
- Basic task management (create, assign, comment, label)
- 500 MB file storage
- Email support (48-hour response time)
- Mobile app access
- Community forum access

### Limitations
- No AI features
- No automation rules
- No timeline or calendar views
- No custom fields
- No integrations (except email)
- No API access
- NovaTech branding on shared views

## Pro Plan

**Price**: $12 per user / month (billed annually) or $15 per user / month (billed monthly)

Best for growing teams that need AI-powered project management and integrations.

### Included Features
- Everything in Free, plus:
- Unlimited team members
- Unlimited active projects
- All board views (Kanban, Timeline, Calendar, Dashboard)
- AI Task Prioritization with daily Priority Digest
- Sprint planning with Capacity Calculator
- 25 NovaFlow automation rules
- All integrations (GitHub, Slack, Figma, etc.)
- Custom fields (up to 20 per project)
- Document Hub with version history
- 10 GB file storage per user
- API access (rate limit: 1,000 requests/hour)
- Priority email support (12-hour response time)
- Export to CSV, JSON, and PDF

### Volume Discounts
- 50-99 users: 10% discount
- 100-249 users: 15% discount
- 250+ users: 20% discount

## Enterprise Plan

**Price**: Custom pricing (contact sales)

Best for large organizations requiring advanced security, compliance, and dedicated support.

### Included Features
- Everything in Pro, plus:
- Unlimited NovaFlow automation rules
- Unlimited custom fields
- AI Risk Prediction and Sprint Forecasting
- Advanced analytics with custom report builder
- Unlimited file storage
- API access (rate limit: 10,000 requests/hour)
- SSO/SAML authentication
- SCIM user provisioning
- Custom data retention policies
- Dedicated customer success manager
- 99.99% uptime SLA
- Phone support (1-hour response time for critical issues)
- Custom onboarding and training
- Quarterly business reviews
- Audit log with 2-year retention
- IP allowlisting
- Custom contract and invoicing

### Enterprise Add-Ons
- **NovaBoard Data Residency**: Choose data storage region (US, EU, APAC). $3/user/month.
- **NovaBoard Advanced Security**: HIPAA compliance, field-level encryption, DLP policies. $5/user/month.
- **NovaBoard Premier Support**: 24/7 phone support, 15-minute response SLA, dedicated support engineer. $8/user/month.

## Billing & Payments

- **Payment Methods**: Credit card (Visa, Mastercard, Amex), ACH bank transfer (US only), wire transfer (Enterprise only).
- **Currency**: Prices listed in USD. Euro and GBP billing available for Enterprise.
- **Upgrades**: Prorated billing when upgrading mid-cycle.
- **Downgrades**: Take effect at the end of the current billing cycle.
- **Cancellation**: Cancel anytime. Data retained for 30 days after cancellation.
- **Education Discount**: 50% off Pro plan for accredited educational institutions.
- **Nonprofit Discount**: 30% off Pro plan for registered 501(c)(3) organizations.
- **Startup Program**: Free Pro plan for up to 10 users for startups with less than $1M in funding. Apply at novatech.example.com/startups.
